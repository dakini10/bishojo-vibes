### Brand Position

8つのFANZA同人/動画アフィリエイトサイトを横断する **Experience Layer**。
各サイトがコンテンツの入り口、Bishojo Vibes がユーザー体験の統一ブランド。

```
Bishojo Vibes (Experience Layer)
├── Shared identity (login, profile, level)
├── Shared economy (Vibe, Pulse)
├── Shared collection (badges, avatars)
└── Shared community (comments, events)

Individual Sites (Content Layer)
├── gero-heaven.com
├── futanari-heaven.com
├── tentacle-heaven.com
├── parody-heaven.com
├── ntr-heaven.com
├── gokkun-heaven.com
├── ishukan-heaven.com
└── saimin-heaven.com
```

## 3. Color System

### Primary Palette

| Role | Color | Hex | Usage |
|------|-------|-----|-------|
| Primary | Lime Green | `#c7f284` | ブランドアクセント、CTA、レベル表示 |
| Secondary | Neon Pink | `#ff69b4` | ハイライト、通知、イベント強調 |
| Background | Deep Navy | `#0a1520` | 既存サイト背景と統一 |
| Surface | Dark Blue | `#101e2c` | カード・モーダル背景 |
| Text Primary | Off White | `#e8e6e3` | 本文テキスト |
| Text Secondary | Muted Gray | `#8a9bb0` | 補助テキスト |

### Color Ratio

ブランド表現における使用比率の目安:

- **Background/Surface**: 70% — ダーク基調を維持
- **Primary (Lime)**: 20% — アクセント・インタラクション要素
- **Secondary (Pink)**: 10% — ハイライト・特別な強調のみ

### Gradients

ブランドグラデーション（ロゴ・特別なUI要素に使用）:

```
BiVi Gradient: #c7f284 → #ff69b4 (Lime to Pink, 135deg)
```

## 4. Tone & Manner

### Voice Character

**Cool & Stylish + Competitive Edge**

洗練されたクールさをベースに、ゲーミングの煽り・挑発を差し込むスタイル。
ユーザーを「参加者」ではなく「プレイヤー」として扱う。

### Tone Spectrum

```
[Casual] -------- [BiVi Zone] -------------- [Formal]
                   ^^^^^^^^^^^
           Cool-Stylish + Gaming Edge
```

### Voice Examples

| Context | Example | Tone |
|---------|---------|------|
| Level up | "Vibe Lv.30 到達。上位5%の領域へ。" | Cool + 煽り |
| Daily login | "Daily Vibe 獲得。streak を切らすな。" | Minimal + 緊張感 |
| Badge earned | "Rare drop. この称号の所持率 2.1%。" | ステータス演出 |
| Gacha | "BVガチャ — 次の一手を引け。" | シンプル・クール |
| Streak broken | "Streak lost. ゼロに戻る。" | 冷たく突き放す |
| Event | "残り72時間。限定バッジ。二度目はない。" | 煽り・緊急性 |
| Welcome | "Your vibe starts here." | スタイリッシュ（短い英語） |
| Profile | "Your vibes, your proof." | 自己表現の肯定（短い英語） |

### Do / Don't (Tone)

| Do | Don't |
|----|-------|
| 短く、断定的に | 長い説明文 |
| 数字で実績を語る（top 5%, 2.1%） | 曖昧な褒め言葉（すごい！、おめでとう！） |
| ユーザーの行動を「記録」として扱う | 過度にフレンドリーな語りかけ |
| 英語のショートフレーズを混ぜる | 絵文字の多用 |
| 静かな緊張感を演出する | テンション高いノリ（！！！連打） |

## 5. Visual Motif (Testing Phase)

最終決定前。2案をテストし、ブランドの視覚言語を確定する。

### Candidate A: Pulse / Waveform (本命)

「Vibes」「Pulse」の世界観に直結する波動・脈動のビジュアル。

- 音波のような波形パターン
- 心拍モニター的なラインアニメーション
- レベルアップ時に波形が増幅するエフェクト
- ロゴの背景・装飾要素として使用

### Candidate B: Spark / Glow

サイバーパンク路線との一貫性を重視した光の粒子・発光表現。

- グロー効果（テキスト・ボーダー周辺の発光）
- パーティクルエフェクト
- ネオンサイン的な表現
- 既存サイトのホバーエフェクトとの親和性

### Decision

LOGO_SPEC.md でプロトタイプを作成し、ビジュアルテストの上で決定。

## Appendix: Design Decisions

| # | Decision | Rationale |
|---|----------|-----------|
| D1 | 文脈別の言語優先 | サイト内は日本語優先（国内ユーザー向け）、SNSは英語優先（グローバル展開の余地） |
| D2 | Lime をプライマリ維持 | 既存8サイトとの視覚的統一。ブランドの連続性 |
| D3 | Pink をセカンダリ | ブランド独自要素として差別化。控えめに使用 |
| D4 | Cool + Gaming Edge トーン | ユーザー層（ゲーマー/オタク）に訴求。安っぽさ回避 |
| D5 | v0.1 は定義のみ | ティザー・SNS開始に十分。UI適用ガイドは v1.0 |
