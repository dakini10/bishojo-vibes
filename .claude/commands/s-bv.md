美少女Vibesプロジェクトの進捗を更新・反映。

## 手順

1. PROGRESS.md を読み込み、行数を確認する
   - 200行を超えていたら警告する（肥大化）
2. 完了タスクと変更内容を PROGRESS.md に記録する
3. CLAUDE.md の Current Status / Implementation Steps を更新する
4. 次のアクション・TODO を明確にする

## PROGRESS.md ルール

- 1エントリ最大10行
- タイトル + 要約（2-3行）+ 更新ファイル + ステータス
- 背景・分析は書かない
- 次のアクションは動詞 + 条件で具体的に
- 未解決のブロッカーがあれば明記

## 読み込むファイル

- docs/PROGRESS.md
- CLAUDE.md
